# AWJ POS Landing Page

صفحة تعريفية عربية خفيفة لمنصة **أوج AWJ POS**، مبنية بـ HTML وCSS وJavaScript فقط، وجاهزة للنشر دون أي عملية Build.

## التشغيل محليًا

افتح ملف `index.html` مباشرة في المتصفح، أو شغّل خادمًا محليًا بسيطًا من مجلد المشروع:

```bash
python3 -m http.server 8000
```

ثم افتح `http://localhost:8000`.

## النشر على GitHub Pages

1. ارفع الملفات إلى مستودع GitHub.
2. من **Settings → Pages** اختر **Deploy from a branch**.
3. اختر الفرع `main` والمجلد `/ (root)` ثم اضغط **Save**.

جميع مسارات الملفات نسبية، لذلك تعمل الصفحة سواء نُشرت على نطاق رئيسي أو داخل مسار مستودع مثل `/landing-page/`.

> نموذج التواصل واجهة تجريبية فقط ولا يرسل أو يخزّن أي بيانات.
